# Service Booking App 🚀

A modern service booking web app built with React, Redux Toolkit, and Tailwind CSS.

## ✨ Features
- Browse services
- Book professionals
- Manage bookings
- Responsive UI
- Developer profile with portfolio & resume

## 🛠 Tech Stack
- React
- Redux Toolkit
- Tailwind CSS
- React Router
